#  Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript.

##  Features
- Search weather by city
- Real-time temperature & humidity
- Clean and responsive UI
- Error handling for invalid cities

##  Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
