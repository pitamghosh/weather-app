# Project Report - Weather Application

## Project Title
Weather Application

## Objective
To build a weather application that shows live weather information using API integration.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## Features
- Search weather by city
- Temperature display
- Humidity display
- Weather condition display
- Error handling

## Development Process
1. Designed UI layout
2. Connected API
3. Fetched weather data
4. Displayed data dynamically
5. Added error handling

## Challenges Faced
- API response handling
- Error handling for invalid cities
- Dynamic UI updates

## Learning Outcome
- API integration
- JSON data handling
- Async JavaScript concepts

## Conclusion
This project improved my practical knowledge of API integration and JavaScript development.
