# Weather Application

A simple and responsive Weather Application built using HTML, CSS, and JavaScript with API integration for real-time weather updates.

## Live Demo
https://pitamghosh.github.io/synent-task6-weather-pitam/

## Project Objective
The objective of this project is to build a weather application that provides real-time weather information based on city search.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## Features
- Search weather by city name
- Real-time temperature display
- Humidity information
- Weather condition display
- Error handling for invalid city names
- Responsive user interface

## Development Process
1. Created the application layout using HTML  
2. Styled the interface using CSS  
3. Connected weather API using JavaScript  
4. Fetched and displayed live weather data  
5. Added error handling
